Energy Label Calculator for properties.

== Description ==
Calculate energy labels for properties using our interactive calculator widget.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/energy-calculator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Energy Calculator screen to configure the plugin
4. Use shortcode [energy_calculator] to display the calculator
